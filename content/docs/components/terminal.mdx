## Usage

```tsx
import Terminal from "@/components/ui/terminal"

export default function Demo() {
  const customCommands = [
    {
      command: "about",
      description: "Learn more about the project",
      action: () => {
        window.open("https://github.com/your-username/your-project", "_blank");
      },
    }
  ];

  return (
    <Terminal 
      commands={customCommands}
      initialMessage="Welcome! Type 'help' to see available commands."
    />
  )
}
```

## Props

| Prop           | Type      | Description                                | Default |
| -------------- | --------- | ------------------------------------------ | ------- |
| className      | string    | Additional classes for the terminal        | -       |
| initialMessage | string    | Initial message shown in the terminal      | "Type 'help' to see available commands" |
| commands       | Command[] | Array of custom commands for the terminal  | []      |

### Command Interface

```typescript
interface Command {
  command: string;      // Command trigger word
  description: string;  // Command description shown in help
  action: () => void;   // Function to execute when command is triggered
}
```

## Built-in Commands

The terminal comes with two built-in commands:

- `help`: Shows all available commands and their descriptions
- `clear`: Clears the terminal history
